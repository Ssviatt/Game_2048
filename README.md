# Game 2048
In this project, I implemented the game 2048 using basic HTML, SCSS, and JS.

### Installation
1. Clone the repo:<br>
  ```sh
  git clone https://github.com/Ssviatt/Game_2048.git
  ```

2. Install NPM packages:<br>
  ```sh
  npm i
  ```
